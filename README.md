# Network Security Scanner

## Overview
This is a lightweight Python tool designed to scan specific IP addresses for open ports. It helps identify active services and potential security entry points in a network environment.

## Key Features
- **Targeted Scanning:** Checks for essential services like SSH (22), HTTP (80), and HTTPS (443).
- **Timeout Management:** Uses socket timeouts to ensure efficient scanning.
- **Service Mapping:** Identifies the common service name associated with each port.

## Technical Skills Demonstrated
- **Python Programming:** Use of the `socket` and `datetime` libraries.
- **Networking:** Understanding of TCP/IP protocols and Port management.
- **Security:** Basic auditing of network exposure.

## How to Run
1. Ensure Python 3 is installed.
2. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Network-Security-Scanner.git`
3. Run the script: `python scanner.py`
